SkyService

PHP Test Task

Створіть PHP-проєкт, який буде реалізовувати API для управління користувачами, постами та коментарями. Проєкт повинен використовувати Composer для управління залежностями та автозавантаження класів.

1. Створіть базу даних MySQL(phpMyAdmin) з трьома таблицями:

Database Tables:

Table Columns
users id, name, email, password, created_at, updated_at
posts id, title, content, user_id, created_at, updated_at
comments id, content, user_id, post_id, created_at, updated_at

2. Створіть моделі (класи) для кожної таблиці:

Models:

  • User
  • Post
  • Comment

3. Створіть контролери для кожної з моделей, які будуть відповідати за відповідні функції:

Controllers:

  • UserController
    • Методи для створення, оновлення, видалення та виведення користувачів.
  • PostController
    • Методи для створення, оновлення, видалення та виведення постів користувачів.
  • CommentController
    • Методи для створення, оновлення, видалення та виведення коментарів.

4. Використовуйте автозавантаження класів для підключення моделей та контролерів до проєкту.

5. Реалізуйте API, яке буде приймати параметр action з вказівкою на виконану дію та інші параметри через $_GET, $_POST.

  • Приклад: api.php?action=getUsers.

6. Налаштуйте composer.json для автозавантаження класів.

7. Необхідні API маршрути

Використати:

  • LEFT JOIN
  • WHERE
  • GROUP BY

Endpoints:

  • GET /api.php?action=getUsers

    Отримати список користувачів.

  • GET /api.php?action=getPosts

    Отримати список постів.

  • GET /api.php?action=getComments

    Отримати список коментарів.

  • GET /api.php?action=getCommentsCount

    Отримати список коментарів для користувачів.

  • GET /api.php?action=getUser&id={id}

    Отримати конкретного користувача за його id.

  • GET /api.php?action=getPost&id={id}

    Отримати конкретний пост за його id.

  • GET /api.php?action=getUserPosts&id={id}

    Отримати список постів конкретного користувача за його id.

  • GET /api.php?action=getPostComments&id={id}

    Отримати список коментарів для конкретного поста за його id.

  • POST /api.php?action=createUser

    Створити нового користувача.

    Параметри: name, email, password

  • POST /api.php?action=createPost

    Створити новий пост.

    Параметри: title, content, user_id

  • POST /api.php?action=createComment

    Створити новий коментар.

    Параметри: content, user_id, post_id

API Routes:

  • GET /api.php?action=getUsers
  • GET /api.php?action=getPosts
  • GET /api.php?action=getComments
  • GET /api.php?action=getCommentsCount
  • GET /api.php?action=getUser&id={id}
  • GET /api.php?action=getPost&id={id}
  • GET /api.php?action=getUserPosts&id={id}
  • GET /api.php?action=getPostComments&id={id}
  • POST /api.php?action=createUser
  • POST /api.php?action=createPost
  • POST /api.php?action=createComment

API Parameters:

  • action
  • id
  • name
  • email
  • password
  • title
  • content
  • user_id
  • post_id
